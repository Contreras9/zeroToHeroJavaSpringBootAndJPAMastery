First Time Git & Github Setup
==============================
1. git init
2. git add .
3. git commit -m "any message"
4. git remote add origin https://github.com/userName/repoName.git
5. git remote set-url origin https://userName:token@github.com/userName/repoName.git
6. git push --set-upstream origin master

Next Time Onwards
==================
1. git pull
2. git status
3. git add .
4. git commit -m "some message"
5. git push
